# Vitality — Mom's Wellness App

A cozy, premium wellness app built with React + Vite.
Deployed as a PWA so it installs to the iPhone home screen.

## What's where
- `src/App.jsx` — the entire app (meals, fitness, wellness, learn, today)
- `src/main.jsx` — React entry point
- `index.html` — the page shell + iPhone home-screen meta tags
- `public/manifest.json` — PWA manifest (icon, name, theme color)
- `public/icon-*.png` — home-screen icons
- `package.json` — dependencies
- `vite.config.js` — build config

## To edit later
1. Open `src/App.jsx` on github.com
2. Click the pencil ✏️ → make your change → commit
3. Vercel auto-deploys in ~60 seconds — refresh her app and it's there
